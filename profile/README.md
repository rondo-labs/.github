## Core Project

### Threader

A desktop-first football analysis workstation designed for modern performance environments.

Threader combines:

- 🎬 **Video Review** — timeline-based match analysis, clips, playlists, tagging  
- 📊 **Data Integration** — event data, tactical labels, custom taxonomies  
- 🧠 **Tactical Intelligence** — semantic structures for patterns, phases, and principles  
- 🤝 **Collaboration** — comments, shared workspaces, review workflows  
- ⚡ **Analyst Speed** — built for real workflows, not just dashboards  

Current focus: creating a faster and more connected alternative to fragmented legacy tooling.
